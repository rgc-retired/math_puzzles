Problem

Position 8 queens on a chessboard such that that number of squares that is not attacked is a maximum

Seems simple enough but the number of combinations is large C(64,8)~4.4 Billion. With reflections, translations, rotations, etc. I am sure the number is much smaller … but there doesn’t seem to be an obvious alorithm to search out just the unique configurations.

  • queens3.py performs exhaustive search

  • queens.tcl provides a board to try the problem manually.

After exhaustive search

The best achievable score is 11. There are 48 configurations that achieve this. Many are reflections or rotations. They have not been removed.
